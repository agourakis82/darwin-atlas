## Methodology

For each sequence, we compare the observed `orbit_ratio` against a null
distribution generated by GC-preserving shuffles. This tests whether the
observed symmetry is significantly different from random expectation.

- **Null model**: 100 GC-shuffled sequences per replicon
- **Test statistic**: z-score = (observed - null_mean) / null_std
- **Significance**: Two-tailed p-value from standard normal distribution
